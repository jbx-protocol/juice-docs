# Contributing to JBX 

JBX has an open proposal process accessible from [Nance](https://nance.app/s/juicebox), and all communications are done publicly within the [Juicebox Discord server](https://discord.gg/juicebox/). There is no core team with outsized power, influential contributors and decision makers build leverage through JBX holdings on their own.
